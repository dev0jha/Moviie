# MovieRecommendationCLI

The **MovieRecommendationCLI** is a command-line application built with Python that helps users discover movies based on their preferences. By leveraging movie datasets and recommendation algorithms, this tool provides personalized movie suggestions directly in your terminal.

## Features

- Command-line interface for ease of use
- Personalized recommendations based on user input/preferences
- Utilizes movie datasets and recommendation algorithms

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dev0jha/Moviie.git
   cd Moviie
   ```
2. **Install required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
   *(Ensure Python 3.x is installed.)*

## Usage

Run the CLI using:
```bash
python main.py
```
